## 项目介绍
输入文字祝福，自动将文字转化为唯美音符展示的趣味小网页项目。
页面风格简约治愈，交互轻柔，氛围感拉满。

灵感参考小红书博主：**@麻省理工rui** 圣诞贺卡创意，在此基础上二次改编为音符祝福卡片。

## 功能特点
- 自定义输入专属文字祝福
- 文字一键转化为动态音符视觉效果
- 轻量纯网页实现，开箱即用
- 简洁美观UI，适配电脑端浏览
